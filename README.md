# R'Home Renov - Portfolio Website

🏠 Site vitrine moderne pour R'Home Renov - Rénovation & Artisanat en Haute-Loire

## 🎯 Caractéristiques

- ✨ Animations 3D fluides avec Three.js
- 🎨 Design moderne et responsive
- 📱 Mobile-first approach
- 🚀 Performance optimisée
- 🎬 Animations Framer Motion
- 🔧 Certification RGE

## 📋 Sections

1. **Hero** - Présentation avec animation 3D
2. **Services** - 3 domaines d'expertise
3. **Projets** - Galerie des réalisations
4. **À propos** - Présentation de l'entreprise
5. **Contact** - Formulaire et informations
6. **Footer** - Navigation et informations

## 🛠️ Stack Technologique

- **React 18** - Framework UI
- **Vite** - Build tool ultra-rapide
- **Three.js** - Graphiques 3D
- **React Three Fiber** - Intégration Three.js
- **Framer Motion** - Animations

## 🚀 Installation

```bash
git clone https://github.com/jovanille43/rhome-renov-portfolio.git
cd rhome-renov-portfolio
npm install
```

## 📦 Lancement

```bash
npm run dev
```

Le site sera disponible sur `http://localhost:3000`
